কিভাবে ব্যবহার করবেন?

  1. GitHub-এ যান: Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. "Generate new token" ক্লিক করুন এবং repo permission দিন
  3. Token টি কপি করে উপরের ফর্মে পেস্ট করুন
  4. Repository owner এবং name দিন (যেমন: username/my-repo)
  5. ফাইল সিলেক্ট করুন এবং আপলোড করুন

⚠️ সতর্কতা: এই token টি কখনও পাবলিক জায়গায় শেয়ার করবেন না। Production এ backend API ব্যবহার করুন।
